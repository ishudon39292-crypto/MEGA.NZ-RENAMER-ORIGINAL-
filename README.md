# 🚀 Mega Cloud Parallel Bulk Renamer Bot

An advanced, high-speed, multi-user parallel Telegram bot built to bulk rename files on MEGA.nz cloud drives efficiently.

## ✨ Features
* 💎 **Premium Diamond Progress Bar:** Real-time visual tracking.
* ⏱️ **Live Math Analytics:** Shows exact Time Elapsed and Estimated Time (ETA).
* 🛑 **Instant Task Termination:** Stop active processes midway with a single click.
* 🔄 **Step-by-Step Security Login:** Secure and interactive user workflow.
* 🚀 **Multi-User Thread Pool:** Runs up to 100 threads in parallel with 0% task loss.

## 🛠️ Commands
* `/start` - Launch the welcome dashboard
* `/login` - Securely connect your MEGA account (Step-by-step)
* `/replace <old_text> <new_text>` - Quick search and replace text in filenames
* `/fullrename <new_name>` - Absolute bulk numbering rename for entire drive
* `/logout` - Wipe local session cache safely
* `/cancel` - Stop the current setup workflow

## ⚙️ Deployment
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Put your `BOT_TOKEN` inside `bot.py`.
4. Run: `python bot.py`
